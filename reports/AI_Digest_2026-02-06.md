DFlash introduces a block diffusion model for drafting during speculative decoding, achieving over 6x lossless acceleration compared to autoregressive decoding and surpassing state-of-the-art speculative decoding methods by more than 2.5x. By using a diffusion model to generate draft tokens in parallel conditioned on the target LLM's hidden features, DFlash bridges the gap between autoregressive and diffusion LMs, mitigating the quality issues typically associated with standalone diffusion LMs. This architecture offers significant benefits for reducing inference latency, lowering serving costs, and democratizing access to large LLMs, making it immediately relevant for production deployments.

*   **Continual Learning:**
    *   **Shared LoRA Subspaces for almost Strict Continual Learning:** A parameter-efficient continual learning method called "Share" learns and dynamically updates a single, shared low-rank subspace for LoRA adapters, achieving up to 100x parameter reduction and 281x memory savings.

*   **Multi-Agent Systems:**
    *   **DyTopo: Dynamic Topology Routing for Multi-Agent Reasoning via Semantic Matching:** A multi-agent framework that dynamically reconstructs a sparse, directed communication graph at each round of reasoning, adapting to the specific needs of the problem-solving stage, improving communication efficiency and reasoning accuracy.
    *   **CommCP: Efficient Multi-Agent Coordination via LLM-Based Communication with Conformal Prediction:** An LLM-based decentralized communication framework for multi-agent embodied question answering that employs conformal prediction to calibrate the confidence of generated messages, enhancing communication reliability and exploration efficiency.
    *   **PhysicsAgentABM: Physics-Guided Generative Agent-Based Modeling:** A framework that shifts inference to behaviorally coherent agent clusters, using state-specialized symbolic agents and a multimodal neural transition model for calibrated cluster-level transition distributions, enabling more scalable and calibrated simulations of complex systems.

*   **Vision Language Models:**
    *   **Can vision language models learn intuitive physics from interaction?:** Explores whether Vision Language Models can learn intuitive physics through interaction with a simulated environment, using reinforcement learning to allow them to "play" with objects and learn their physical dynamics.

*   **AI Agent Security:**
    *   **Prompt Injection Detection & Mitigation API:** A tool that hardens AI agents against manipulation and prompt injection by dynamically stripping prompt injections while preserving captchas.

*   **VR Simulation:**
    *   **Learning Event-Based Shooter Models from Virtual Reality Experiments:** Develops a data-driven discrete-event simulator that models shooter movement and in-region actions as stochastic processes learned from participant behavior in VR studies to evaluate school security measures.
