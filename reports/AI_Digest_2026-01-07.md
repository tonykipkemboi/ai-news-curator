The "Software-Defined Agentic Serving" paper introduces a new framework inspired by Software-Defined Networking (SDN) to dynamically control agent communication and optimize serving performance in multi-agent LLM pipelines. This is architecturally significant because it moves beyond static agentic pipelines to a model of dynamic resource allocation and routing. The vision of declarative control over agent communication further promises to simplify the configuration and management of complex agentic pipelines.

**A. Agentic Memory and Reasoning**

*   **MAGMA: A Multi-Graph based Agentic Memory Architecture for AI Agents:** Focuses on improving long-context reasoning in LLM agents by representing memory items across orthogonal semantic, temporal, causal, and entity graphs. This allows for query-adaptive selection and structured context construction, addressing the challenge of maintaining long-term coherence and transparency in agent reasoning.
*   **InfiAgent: An Infinite-Horizon Framework for General-Purpose Autonomous Agents:** This framework addresses the problem of context growth in long-horizon autonomous agents by externalizing persistent state into a file-centric state abstraction. This enables stable task completion over extended execution horizons and reduces cognitive load on the LLM.

**B. LLM Reasoning and Benchmarking**

*   **STRReasoner: Empowering LLMs for Spatio-Temporal Reasoning in Time Series via Spatial-Aware Reinforcement Learning:** This enhances LLMs' ability to perform spatio-temporal reasoning in time series data through a novel benchmark (ST-Bench), architecture (STReasoner), and reinforcement learning algorithm (S-GRPO). This allows LLMs to answer complex questions about "what happened, where, when, and why" in spatio-temporal systems.
*   **UltraLogic: Enhancing LLM Reasoning through Large-Scale Data Synthesis and Bipolar Float Reward:** Introduces a framework for automated generation of high-quality reasoning data and a Bipolar Float Reward (BFR) mechanism for reinforcement learning. This addresses the challenge of data scarcity and improves reasoning abilities in LLMs.
*   **The Sonar Moment: Benchmarking Audio-Language Models in Audio Geo-Localization:** Introduces AGL1K, the first audio geo-localization benchmark for audio language models (ALMs). This provides a standardized way to evaluate ALMs' ability to infer the geographic origin of a given audio signal, enabling applications in fact-checking, investigations, and public safety.

**C. Multi-Agent Communication and Interpretability**

*   **Automated Semantic Rules Detection (ASRD) for Emergent Communication Interpretation:** Addresses the interpretability gap in emergent communication within multi-agent systems by automatically extracting patterns from agent messages and relating them to input data attributes. This is crucial for debugging, monitoring, and ensuring safety in multi-agent systems.

**D. AI in Education**

*   **MalruleLib: Large-Scale Executable Misconception Reasoning with Step Traces for Modeling Student Thinking in Mathematics:** Provides a framework for translating documented misconceptions into executable procedures, enabling scalable supervision and controlled evaluation of student modeling in mathematics. This is valuable for building educational AI systems that can diagnose student misconceptions and provide targeted feedback.
